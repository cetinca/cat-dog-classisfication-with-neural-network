Description

We have trained the base model on the training set tracking its performance on the validation set but our goal is to use our model to make predictions.

In this stage, use the model saved in the previous stage to make predictions on the test set that you generated in the first stage.

As our model outputs probability values in a two-column matrix (with two probabilities predicted for each observation), use the numpy's function argmax to get an array of the actual classes (0 or 1) based on the maximum probability values.

Objectives

In this stage, make predictions with your base model. Your program should:

  1. Import load_model from tensorflow.keras.models
  2. Load stage_two_model.h5 from SavedModels directory
  3. Make predictions on the test set using the predict method and apply the argmax function
  4. Save the predictions as a pickle file named stage_three_history in the SavedHistory directory

Our goal at this stage is to ensure that the difference between the train and test accuracies, and the difference between the validation and test accuracies do not exceed 15%.
